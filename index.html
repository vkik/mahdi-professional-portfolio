Truss Analysis Using the Stiffness Method

A comprehensive expansion into matrix displacement methodologies, thermal effects, and space trusses [cite: 1, 1106, 1290].

14.1 Fundamentals of the Stiffness Method

The stiffness method is a displacement method [cite: 9]. It can be used to analyze both statically determinate and statically indeterminate structures, giving it a universal advantage over the flexibility method [cite: 12].

  • Kinematic Indeterminacy: Unconstrained displacements represent the primary unknowns [cite: 42].
  • Degrees of Freedom (DOF): Generally, there are two degrees of freedom for each joint [cite: 43].
Code Numbering Rule:
Lowest numbers = Unknown displacements (unconstrained DOFs) [cite: 48].
Highest numbers = Known displacements (constrained DOFs) [cite: 48].

14.6 Application & Matrix Partitioning

Once assembled, the structure stiffness equation is Q = K D [cite: 445].

By partitioning the matrices into known (k) and unknown (u) components [cite: 450], we solve directly for unconstrained displacements.

Qk = K11 Du + K12 Dk [cite: 455]

Assuming Dk = 0 [cite: 459]:
Du = K11-1 Qk [cite: 462]

14.7 Nodal Coordinates for Inclined Supports

When a truss is supported by a roller on an incline, constraint cannot be defined using a single horizontal/vertical global system [cite: 883].

We use nodal coordinates (x'', y'') located at the inclined support [cite: 886].

The stiffness matrix expands to incorporate direction cosines mapping local (x'), global (x, y), and nodal (x'', y'') axes [cite: 930, 931].

14.8 Thermal Changes & Fabrication Errors

Modifications required when members change length due to thermal expansion or fabrication errors. This is crucial for indeterminate trusses [cite: 1107, 1111].

Thermal Fixed-End Forces:
(qN)0 = AEαΔT [cite: 1116]
(qF)0 = -AEαΔT [cite: 1116]
Fabrication Fixed-End Forces:
(qN)0 = AEΔL / L [cite: 1122]
(qF)0 = -AEΔL / L [cite: 1122]
Modified Stiffness Equation:
Q = K D + Q0 [cite: 1130]
Where Q0 is the column matrix of initial fixed-end forces [cite: 1132].
